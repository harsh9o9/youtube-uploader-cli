# YouTube Uploader CLI (In-Progress)

A command-line interface (CLI) tool for uploading videos to YouTube using the YouTube Data API v3. This project is designed to simplify the process of managing and uploading videos to your YouTube account from the terminal.

## Features

- Upload videos to YouTube.
- Create and manage playlists.
- Add videos to existing playlists.

## Requirements

- Node.js v20.16.0 or higher
- Google Cloud Project with YouTube Data API v3 enabled
- OAuth 2.0 credentials (Client ID and Client Secret)

## Setup

```bash
git clone https://github.com/your-username/youtube-uploader-cli.git
cd youtube-uploader-cli
npm start
```
